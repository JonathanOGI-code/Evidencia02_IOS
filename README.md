# Evidencias02  
Portafolio de Actividades – Desarrollo de Aplicaciones Móviles (iOS – Swift)

Este repositorio contiene el proyecto **Evidencias02**, el cual integra como **submódulos** los proyectos desarrollados durante las Actividades 5, 6, 7 y 8 del módulo de Desarrollo de Aplicaciones Móviles para iOS.  
Cada actividad representa un avance en el manejo de interfaces, navegación y componentes esenciales dentro de Xcode usando Swift y SwiftUI/UIKit.

A continuación se presentan los proyectos incluidos, junto con una breve reseña y mi experiencia realizándolos.

---

## Proyectos Incluidos

### 1. Actividad 5 – Menú Lateral
**Descripción:**  
Proyecto enfocado en implementar un **Side Menu** dentro de una aplicación iOS.  
Se trabajó con el uso de librerías o estructuras personalizadas para mostrar un menú deslizable desde el lateral y controlar la navegación entre pantallas.

**Experiencia:**  
Fue mi introducción al concepto de menús laterales en iOS. Aprendí sobre la integración de controladores, manejo de vistas y animaciones básicas. Además, reforcé mi comprensión sobre el flujo de navegación en aplicaciones profesionales.

---

### 2. Actividad 6 – Menú de Opciones en la App
**Descripción:**  
Aplicación destinada a implementar un menú de opciones utilizando elementos como `Toolbar`, `NavigationBarItems` o menús contextuales dentro de SwiftUI o UIKit.  
Incluye botones de acciones como configuraciones, ayuda o acciones rápidas según el diseño del ejercicio.

**Experiencia:**  
Esta actividad me ayudó a entender cómo gestionar acciones desde la barra superior, cómo organizar las opciones del usuario y cómo responder a eventos mediante `@Actions`, `Button`, `NavigationLink` o controladores tradicionales.

---

### 3. Actividad 7 – My Movies
**Descripción:**  
Proyecto centrado en mostrar un listado de películas con datos personalizados e interfaces construidas mediante **List**, **TableView** o **CollectionView**, según el enfoque del desarrollo.  
Se trabajó con modelos de datos, estructuras personalizadas y vistas más elaboradas.

**Experiencia:**  
Este proyecto fue uno de los más completos. Fortaleció mis habilidades para crear vistas dinámicas, gestionar datos, trabajar con estructuras `Movie`, así como aprender a personalizar celdas y vistas. Me permitió practicar la separación de lógica y vista.

---

### 4. Actividad 8 – Listados
**Descripción:**  
Aplicación orientada a trabajar con diferentes tipos de listados y cómo gestionarlos en iOS.  
Se exploraron alternativas como `List` de SwiftUI, `UITableView` o `UICollectionView` dependiendo del enfoque seleccionado.

**Experiencia:**  
Reforcé conceptos importantes sobre manejo de datos, reutilización de vistas y el manejo eficiente de listados. También comprendí mejor cómo funcionan los ciclos de actualización en SwiftUI y UIKit.

---

## Conclusión General
El proyecto **Evidencias02** reúne las actividades esenciales de esta etapa del curso, integradas mediante **submódulos de Git**.  
Cada actividad aportó conocimientos clave sobre el desarrollo de interfaces, navegación y manejo de información dentro del desarrollo móvil en iOS con Swift.

Este repositorio sirve como evidencia del progreso logrado y como referencia para futuros proyectos.
